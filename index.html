Find Where to Watch YOUR Team

Fan Watch was created in order for displaced fans to know where to go watch their team. There are a lot of factors that are important to a hardcore fans like -- will the sound be on? do lots of fans gather? is the food good?

What's the 'Sitch?

Social Superhero was created for children with autism and the professionals who work with them. The app serves as a trusty sidekick, easily accessible when social situations seem insurmountable. Autism professionals can Create, Read, Update, and Delete situations, tailored to each child.

Version 2.0

In the next version the version I will add security and secure log in. The child will have access to the situations, but they will be read only. The Autism professional will have an Adminstrative log-in. They will be able to have full control over setting the child's situations for a given week.

The Tech

  • MongoDB
  • Express
  • Node.js
  • React
  • JavaScript
  • Custom CSS

Community-based Backcounty Skiing

Backcountry Buddy's goal is to create a comunity as a resource for backcountry skiing. Users wil post their favorite spots and educate others about the experience. The price to ski and snowboard has been increasing exponentially over the last 20 years, so more and more skiers are turning towards alternatives.

Version 2.0

In the next version, I will add user authentication and security. Users will be able to view all posts but only edit and delete their own.

The Tech

  • MongoDB
  • Express
  • Node.js
  • Handlebars
  • Materialize for CSS

Jeopardy!

Tasked with creating a functional Jeopardy game, I was able to style a board and create user interaction. Using modals and multiple choice I was able to ask for users to answer questions specific to a point total and category. In the bottom right the score board accurately adds and subtracts points based on user success.

Version 2.0

I would like a more user-friendly mobile version and Ideally each new game would truly be a new game. I'd also like to add a timer on final jeopardy. Finally, a double-jeopardy round and allowing users to play each other.

The Tech

  • HTML
  • JavaScript
  • CSS
  • jQuery
  • Materialize
